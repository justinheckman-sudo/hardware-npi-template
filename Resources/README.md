# Resources

This directory contains templates, reference documentation, and automation quick references.

## Directory Structure

```
Resources/
├── Templates/            # Document templates for various types
└── Automations/          # Automation documentation and quick references
```

## Templates

**Available templates:**
- `daily-build-report-template.md` - Daily build status report
- `decision-document-template.md` - Formal decision documentation
- `risk-document-template.md` - Risk register entry

**How to use templates:**
1. Copy template from `Resources/Templates/`
2. Name following vault conventions
3. Fill in all sections
4. Save in appropriate directory
5. Link from related documents
